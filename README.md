# AI Smart Home Automation

This project is an AI-powered smart home automation system that enables users to control multiple household appliances using natural language commands. The system uses an AI chatbot to understand user instructions and toggle devices in real time, offering a simple, smooth, and interactive automation experience.

## Key Features
- Control 10+ simulated home appliances through chat commands  
- 85%+ accuracy in command recognition  
- Real-time device toggling with optimized logic  
- 40% reduction in false responses  
- Clean and user-friendly interaction flow  

## Technologies Used
- Python / JavaScript  
- Natural Language Processing (NLP)  
- Virtual device simulation for home appliances
